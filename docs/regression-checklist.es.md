3. Inventario carga y el título es "Products".

7. Navegación atrás/adelante no rompe la sesión de forma rara.
8. API: `GET /posts/1` devuelve 200 y campos esperados.
9. API: validación de JSON Schema para post pasa.
10. API: data-driven posts (ids 1,2,3) pasa.
